Listas

Não Ordenadas

As lista podem ser ciradas a patir do comando, tag, "ul" e logo após o "li". Como mostrado a baixo:

  • dois
  • tres

Repare que ess lista ficou com quadradinho, isso se dá pela expressão <ul type=square">, Quisermos uma lista com redondos fazemos normalmente.

Ordenadas

Podemos criar uma lista com letras e numeros.

Letras
  1. um
  2. Dois
  3. três

Isso se dá pela marcação "ol type="A" start="1"

Números
  1. A
  2. B
  3. C
Isso se dá pela marcação "ol type="1" start="1". Se substituir o número no start, a lista começará pelo número que lá estiver.

Juntando Listas

  1. Aula
    1. Seção um
    2. Seção dois
  2. Aula
    1. Seção um
    2. Seção Dois
